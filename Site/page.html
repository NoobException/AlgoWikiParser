Przeszukiwanie wszerz

O algorytmie
Przeszukiwanie wszerz (albo też BFS, z angielskiego Breadth-first search) jest algorytmem, który pozwala w czasie liniowym przejrzeć graf. Ma on tę przyjemną własność, że w grafie nieważonym odwiedza wierzchołki w kolejności rosnącej od grafu.
Pseudokod
Algorytm BFS jest implementowany przy użyciu kolejki, do której wrzucamy kolejne wierzchołki. 
                
array visited;
array distance;
queue nodes_to_visit;
nodes_to_visit.push(source);
distance[source] = 0;
visited[source] = true;
while (not nodes_to_visit.empty())
{
    current_node = nodes_to_visit.front();
    nodes_to_visit.pop();
    for (neighbor in get_neighbors(current_node))
    {
        if (visited[neighbor])
            continue;
        visited[neighbor] = true;
        distance[neighbor] = distance[current_node] + 1;
        nodes_to_visit.push(neighbor);
    }
}
